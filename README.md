Spring Boot:
Spring Boot is a framework for building Java applications by:
* Removing complex configuration
* Providing embedded servers(Tomcat,Jetty)
* Offering Production Ready features(metrics,health,etc).

Key Features:
* Auto Configuration -> Automatically configures beans.
* Starter Dependencies -> Predefined dependencies bundles.
* Embedded Server -> No need for external deployment.
* Spring Boot CLI -> Quick app creation.

Spring Data Persistence:
Spring Data simplifies data access by reducing boilerplate code.

Instead of writing SQL,you define interface and Spring handles implementation.

Benefits:
* No DAO implementation required.
* Build in CRUD operations.
* Pagination and sorting Support.
* Integration with multiple databases (MYSQL,H2,POSTGRESQL,MONGODB).

JPA(Java Persistence API)
JPA is a JAVA specification for managing relational data in JAVA.
